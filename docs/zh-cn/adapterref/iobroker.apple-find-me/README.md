# Iobroker.apple-find-me
## IoBroker 的 Apple-find-me 适配器
Apple Find Me Connector 是一个 ioBroker 适配器，用于获取已连接 Apple 设备的当前位置和其他指标。

它无需 2-Factor-Authentication (2FA) 即可工作并检索所有连接的设备。
