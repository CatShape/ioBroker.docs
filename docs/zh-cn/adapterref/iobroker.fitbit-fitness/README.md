# Iobroker.fitbit-fitness
## IoBroker 的 FITBIT 适配器
适用于 fitbit 设备的适配器 此适配器将 fitbit 数据检索到 iobroker。从@GermanBluefox 从他的 fitbit-api (great thx) 移植到新的 iobroker 模板结构中并添加了一些新功能

＃＃ 特征
- 身体、活动、食物、睡眠单独检索
- 在几分钟内刷新以持续检索数据
- 每天可以加载 2 次睡眠记录，以减少 API 调用

## 已知错误
- 目前没有
